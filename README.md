# Broken Link Checker (Cypress Automation)

## Description
Automated tool to detect and report broken website links using Cypress. Test scripts are written in JavaScript, executed with Node.js, and all run results are stored in Cypress Cloud for easy access and analytics.

## Features
- Scans and tests links on a target web page
- Identifies broken links (error status codes)
- Records results in Cypress Cloud

## Technology Stack
- Cypress
- JavaScript
- Node.js
- Cypress Cloud
- Visual Studio Code
