Project Title
Broken Link Checker (Cypress Automation)

Description
This project is an automated tool built with Cypress that checks for broken hyperlinks on a website. The tests are written in JavaScript and executed in Node.js. Test run results are recorded and managed in Cypress Cloud for easy analysis and sharing.

Features
Automatically visits a target webpage and scans links

Detects and reports broken (HTTP 400+) links

Stores all test runs in Cypress Cloud for persistent access

Technology Stack
Cypress

JavaScript

Node.js

Cypress Cloud

Visual Studio Code

How to Use
Clone this repository.

Run npm install to install dependencies.

Edit the test spec in cypress/e2e/ to set your target URL if needed.

Use npx cypress open for interactive testing, or npx cypress run --record to save results in Cypress Cloud.

Cypress Cloud Integration
This project is integrated with Cypress Cloud. All test results are automatically saved to the cloud when run with the --record flag. You can view detailed logs and analytics in your Cypress Cloud dashboard.
